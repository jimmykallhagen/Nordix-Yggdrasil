# Shader system

## Norsix Yggdrasil - post process shader system with Nordix shader manager

- I have developt a shader system that runs on top of every game, video and desktop. in the entire desktop.
- 12-20 professional shaders are ready.
- system for adding shaders in a toggle list is already done
-  GUI nordix shader manager for selecting which shaders to include in the toggle list is deno.
- gui can also adjust all shaders settings


> **everything is integrated in Yggdrasil and a user only needs to press ALT + CAPS to activate and toggle between shaders**
> **Shaders with techniques such as: adobe photoshops clarity with gussian blurr, real amd fidelityFX rcas and cas in a special pipeline system**
> **which I call the Nordic Smart Sharpe "Hack" to enable higher sharpness before flicker occurs, extreme sharpeneing if you want that**
> **the classic tone maps and many more**
> **vivid shaders for the ones that has HDR monitor**
> **CTR shaders with the besel that makes it look like you have a fat old tv, borrowed från cool-retro-terminal**
